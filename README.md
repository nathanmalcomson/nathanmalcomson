## [nathanmalcomson.github.io](https://nathanmalcomson.github.io/)

## How to run locally:
1. Run `git pull`
2. Run `npm run start`
3. Go to http://localhost:8080/

## How to update pages
1. Run `git pull`
2. Edit text in `{page-i-want-to-edit}.html`
3. Run `npm run build`
4. Run `git commit -am "Some message describing what I changed"`
5. Run `git push`
